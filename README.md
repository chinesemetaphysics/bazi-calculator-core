# BaZi Calculator Core

Shared calculation library for Chinese Metaphysics applications.

## Features

- **Four Pillars (BaZi)** - Year, Month, Day, Hour pillar calculations
- **Flying Stars (Xuan Kong)** - Annual, Monthly, Daily, Hourly flying stars
- **Eight Mansions (Ba Zhai)** - Kua number and favorable directions
- **Solar Terms** - Algorithmic calculation (VSOP87)

## Usage via CDN

```html
<!-- Load individual modules -->
<script src="https://cdn.jsdelivr.net/gh/chinesemetaphysics/bazi-calculator-core@v1.0.0/src/constants.js"></script>
<script src="https://cdn.jsdelivr.net/gh/chinesemetaphysics/bazi-calculator-core@v1.0.0/src/solar-terms.js"></script>
<script src="https://cdn.jsdelivr.net/gh/chinesemetaphysics/bazi-calculator-core@v1.0.0/src/pillars.js"></script>
<script src="https://cdn.jsdelivr.net/gh/chinesemetaphysics/bazi-calculator-core@v1.0.0/src/flying-stars.js"></script>

<!-- Or load bundled version -->
<script src="https://cdn.jsdelivr.net/gh/chinesemetaphysics/bazi-calculator-core@v1.0.0/dist/bazi-calculator.js"></script>
```
